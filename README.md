# Roblox Custom Chat UI

A simple Roblox Studio LocalScript that adds:
- A chat toggle button
- Black chat bubbles
- White text
- Auto-sizing message bubbles

## Installation
1. Open Roblox Studio
2. Place the script in:
   StarterPlayer > StarterPlayerScripts
3. Run the game

## Notes
This is a UI demo for learning and prototyping.
